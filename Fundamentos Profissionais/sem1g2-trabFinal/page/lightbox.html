jQuery Lightbox


Com certeza todos já viram em algum site, um efeito quando se clica nas imagens e sobrepõe uma imagem á página.

Em quase todos os sites em que aparece este efeito, recorrem ao lightbox, que é um script em javascript que cria este efeito quando se clica numa imagem que linka outra, por ex um thumbnail que é um link para a imagem em tamanho real, em que a segunda aparece sobreposta á página mostrada.

Isso acontece por que na Thumbnail que tem o link para a imagem em tamanho real é adicionado o atributo data-lightbox="lightbox"

Para que isto funcione é necessário acrescentar no header da página a referencia ao ficheiro onde está o o script do lightbox e suas dependencias e também do css usado pelo lightbox.

Você pode fazer download da biblioteca Lightbox, neste endereço: lokeshdhakar.com/projects/lightbox2
