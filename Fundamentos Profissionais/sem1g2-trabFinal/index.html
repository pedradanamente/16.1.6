Este site é um trabalho final da disciplina Fundamentos Profissionais com o professor Taciano Balardin.

Tende propósito aplicar o conteúdo de tecnologia html, javascript e css aprendido em aula, aprofundando na tecnologia e chegando ao HTML5 como também trazendo o uso de framework para uma determinada parte do site mostrando uma maneira mais produtiva de se desenvolver algo.


A codificação UTF-8 foi usada na página por motivo pessoal



Segue abaixo a estrutura de HTML5 utilizada na página:


	<!DOCTYPE html>

	<html lang="pt-br">

	<head>

		<meta charset="UTF-8">

		<script src="js/script.js"></script>

		<link rel="stylesheet" type="text/css" href="css/estilos.css">

		<title> </title>

	</head>

	<body>


	</body>

	</html>
